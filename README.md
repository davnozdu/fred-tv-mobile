# Smotrim CZ Player

**IPTV-плеер для Android и Android TV** — простой, быстрый и удобный с пульта.
Заточен под ТВ-приставки (Mi TV Stick, Amlogic-боксы и т.п.) и одинаково
хорошо работает на телефонах.

Пакет: `cz.smotrim.player` · Языки интерфейса: русский, английский, украинский
(берётся из системного).

---

## Что это и для кого

Плеер для просмотра IPTV по вашему плейлисту: эфирные каналы, программа
передач (EPG), архив (catchup), категории, избранное и история. Сделан так,
чтобы пожилому человеку было удобно управлять обычным пультом: крупный текст,
понятная навигация, контрастное выделение, листание каналов кнопками.

## Возможности

- **Плейлисты:** M3U (файл), M3U по ссылке, Xtream Codes, **HLS-PROXY**
  (локальный прокси). Можно добавить **несколько плейлистов** и переключаться
  между ними.
- **Категории с иконками:** каналы группируются по категориям из плейлиста,
  каждой подбирается своя иконка и цвет (новости, кино, спорт, страны и т.д.).
- **Программа передач (EPG):**
  - бегущая строка «сейчас идёт» под каждым каналом;
  - сетка вещания в стиле Plex с навигацией по блокам;
  - текущая передача бегущей строкой прямо в плеере.
- **Архив / catchup:** перемотка к прошедшим передачам (Flussonic), на 1 день
  или расширенный до 7 дней.
- **Плеер:** пауза/плей, перемотка ±5/±30 сек, выбор соотношения сторон и
  аудиодорожки, листание каналов кнопками вверх/вниз (или CH+/CH−),
  авто-переподключение при зависании потока.
- **Родительский контроль:** PIN-код на категорию — без кода она не
  открывается и не попадает в листание/сетку.
- **Скрытие категорий:** лишние категории можно убрать из меню.
- **Удобство для ТВ-приставок:**
  - «Продолжать воспроизведение» — если приставку выключили на канале, при
    включении он сам продолжит играть;
  - «Автозапуск при загрузке» — запуск приложения и нужного канала/категории
    после включения приставки;
  - «Вы ещё смотрите?» — при долгом бездействии плеер вежливо переспросит и
    поставит на паузу, чтобы не крутить поток впустую.
- **Автообновление:** проверка новых версий на GitHub и установка в один шаг
  (многопоточная загрузка APK).

## Как это работает

1. **Добавляете плейлист** через мастер настройки (тип источника → данные).
   Для HLS-PROXY достаточно указать IP, порт и имя плейлиста — плеер сразу
   показывает статус сервера и, при необходимости, предложит установить
   компаньон-приложение HLS-PROXY.
2. **Каналы группируются по категориям**, подтягиваются логотипы и программа
   передач из EPG (по умолчанию `epg.one`; для длинного архива —
   `iptvx.one`).
3. **Смотрите**: выбираете канал, листаете пультом, открываете сетку программы
   или архив. EPG разбирается и кэшируется в фоне, поэтому интерфейс и видео не
   тормозят даже на слабых приставках.

## Технологии

- **Flutter** (Dart), единый APK для всех ABI + отдельные сборки
  `arm64-v8a` / `armeabi-v7a`.
- **better_player_plus** (ExoPlayer / Media3) — движок воспроизведения.
- **SQLite** — хранение каналов, источников и настроек.
- EPG (XMLTV) разбирается в фоновом изоляте с кэшем в памяти и на диске.

## Сборка

Сборка идёт на **GitHub Actions** при создании тега `vX.Y.Z`. В релиз
публикуются:

- `app-release.apk` — универсальный (все ABI), используется для автообновления;
- `app-arm64-v8a-release.apk` — 64-битные устройства;
- `app-armeabi-v7a-release.apk` — 32-битные устройства (подходит большинству
  ТВ-приставок).

## Установка

Скачайте APK из раздела [Releases](../../releases) и установите на устройство
(разрешите установку из неизвестных источников). На большинстве ТВ-приставок
берите `armeabi-v7a` или универсальный `app-release.apk`. Дальше приложение
обновляется само.
